# Agent Instructions

Use the repo-local skills before making changes:

- For visual, typography, layout, color, responsive, or component changes, read `.codex/skills/readwise-ghost-theme-design-system/SKILL.md`.
- For theme ZIP, release artifact, or GitHub Actions packaging changes, read `.codex/skills/readwise-ghost-theme-packaging/SKILL.md`.

Generated files matter in this repo. If Sass or JavaScript source changes, run `npm run build` and include the generated `assets/css/style.css` or `assets/js/script.js` changes.
